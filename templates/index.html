O que é Python?

Uma Linguagem de Programação Versátil

Python é uma das linguagens de programação mais populares e versáteis do mundo. Criada com foco na legibilidade do código e simplicidade, ela permite que desenvolvedores expressem ideias complexas em poucas linhas, tornando-a ideal para iniciantes e profissionais experientes.

Sua popularidade vem da sua capacidade de ser usada em diversas áreas, desde desenvolvimento web e análise de dados até inteligência artificial e automação.

A História do Python: Da Origem à Onipresença

O Python foi criado por Guido van Rossum no final dos anos 1980, no Centrum Wiskunde & Informatica (CWI) na Holanda, como um sucessor da linguagem ABC. Van Rossum buscava uma linguagem que fosse fácil de usar, expressiva e poderosa, mas sem as complexidades de linguagens como C ou Pascal.

O nome "Python" foi inspirado no grupo humorístico britânico Monty Python, do qual Guido era fã, e não na cobra. A primeira versão oficial, Python 0.9.0, foi lançada em 1991, marcando o início de uma jornada que levaria a linguagem a se tornar uma das mais influentes do século XXI.

Ao longo dos anos, o Python evoluiu através de várias versões significativas, como a Python 2.x (que dominou por muitos anos e ainda é encontrada em sistemas legados) e a transição para a Python 3.x em 2008, que trouxe melhorias e uma base mais limpa para o futuro da linguagem. Hoje, o Python 3.x é a versão padrão e ativa, com uma comunidade vibrante e um ecossistema de bibliotecas e frameworks vastíssimo.

Exemplos Fundamentais de Código Python

Vamos explorar alguns dos conceitos mais básicos e amplamente usados em Python:

1. Saída de Dados: A Função print()

A função print() é fundamental para exibir informações na tela (console). É a primeira coisa que muitos aprendem em Python.

print("Olá, Mundo!") # Exibe a string "Olá, Mundo!"
nome = "Python"
print(f"Bem-vindo ao {nome}!") # Exibe uma string formatada
2. Variáveis e Tipos de Dados

Variáveis são usadas para armazenar valores. Python é dinamicamente tipado, o que significa que você não precisa declarar o tipo da variável.

idade = 30              # Inteiro
altura = 1.75             # Ponto flutuante
nome_completo = "Ana Silva"  # String
is_estudante = True     # Booleano

print(idade)
print(nome_completo)
3. Condicionais: if, elif, else

Permitem que seu programa tome decisões com base em condições. O código só será executado se uma condição for verdadeira.

nota = 75

if nota >= 70:
    print("Aprovado!")
elif nota >= 50:
    print("Recuperação.")
else:
    print("Reprovado.")
4. Laços de Repetição: for e while

Usados para executar um bloco de código repetidamente. O for é ótimo para iterar sobre sequências, e o while para repetir enquanto uma condição for verdadeira.

Exemplo com for:
frutas = ["maçã", "banana", "cereja"]
for fruta in frutas:
    print(f"Eu gosto de {fruta}.")

# Loop com range (para um número específico de repetições)
for i in range(5): # Repete 5 vezes (0 a 4)
    print(f"Contagem: {i + 1}")
Exemplo com while:
contador = 0
while contador < 3:
    print(f"Contador é {contador}")
    contador += 1 # Equivalente a contador = contador + 1
5. Funções

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas ajudam a organizar o código e torná-lo mais modular.

def saudar(nome):
    """Esta função cumprimenta uma pessoa."""
    print(f"Olá, {nome}!")

saudar("Maria")   # Chama a função
saudar("João")
