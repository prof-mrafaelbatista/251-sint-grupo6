Funções e Procedimentos em Python

Conceito

Em programação, funções e procedimentos são blocos de código reutilizáveis que realizam uma tarefa específica. Em Python, ambos são implementados usando a palavra-chave def. A principal diferença conceitual é que uma função geralmente retorna um valor, enquanto um procedimento executa uma ação sem necessariamente retornar algo (embora em Python, uma função sem return explicitamente retorna None).

  • Função: Um bloco de código nomeado que pode receber argumentos, realizar operações e retornar um valor.
  • Procedimento: Um bloco de código nomeado que pode receber argumentos e realizar uma ação, mas **não retorna um valor** (ou retorna None implicitamente).

Aplicação

Funções e procedimentos são fundamentais para:

  • Reutilização de Código: Evitar a repetição de código, tornando o programa mais conciso e fácil de manter.
  • Modularização: Dividir um programa grande em partes menores e mais gerenciáveis.
  • Abstração: Esconder os detalhes complexos de uma operação, permitindo que o programador se concentre no "o quê" em vez do "como".
  • Organização: Estruturar o código de forma lógica e legível.
  • Testabilidade: Facilitar o teste de partes individuais do programa.

Exemplos de Código

Exemplo 1: Função Simples (com retorno) 

def saudar(nome):
    "Esta função retorna uma saudação personalizada."
    return "Olá, " + nome + "!"

mensagem = saudar("Alice")
print(mensagem)

print(saudar("Bob"))

A função saudar recebe um argumento nome e retorna uma string formatada.

Exemplo 2: Procedimento (sem retorno explícito)


def mostrar_informacao(produto, preco):
    "Este procedimento imprime informações sobre um produto."
    print("Produto:", produto)
    print("Preço:", preco)

mostrar_informacao("Livro", 49.90)
print("\nOutro produto:")
mostrar_informacao("Caneta", 2.50)

O procedimento mostrar_informacao executa a ação de imprimir, mas não retorna nenhum valor para quem o chamou.

Exemplo 3: Função com múltiplos argumentos e condicional


def calcular_media(n1, n2, n3):
    "Calcula a média de três notas e retorna o status."
    media = (n1 + n2 + n3) / 3
    print("Média:", media)
    if media >= 7:
        return "Aprovado"
    else:
        return "Reprovado"

status_aluno1 = calcular_media(8, 7, 9)
print("Aluno 1:", status_aluno1)

status_aluno2 = calcular_media(5, 6, 4)
print("Aluno 2:", status_aluno2)

Funções podem combinar lógica condicional e retornar resultados baseados nela.
