Estruturas de Repetição em Python

Conceito

As estruturas de repetição (ou laços, loops) permitem que um bloco de código seja executado múltiplas vezes. Elas são essenciais quando você precisa repetir uma ação sem ter que escrever o mesmo código várias vezes. As principais estruturas de repetição em Python são for e while.

Aplicação

Estruturas de repetição são usadas para:

  • Iterar sobre Coleções: Percorrer itens de listas, tuplas, strings ou dicionários.
  • Processamento Contínuo: Executar uma ação repetidamente até que uma condição específica seja satisfeita (ex: um menu que continua até o usuário digitar 'sair').
  • Cálculos Iterativos: Realizar operações que exigem repetição (ex: somar todos os números em uma faixa).
  • Automação: Automatizar tarefas repetitivas.

Exemplos de Código

Exemplo 1: Estrutura for (Iterando em uma lista)


frutas = ["maçã", "banana", "cereja"]

for fruta in frutas:
    print(fruta)

Este laço for percorre cada item da lista frutas e imprime o nome de cada uma.

Exemplo 2: Estrutura for (Usando range())


# Imprime números de 0 a 4
for i in range(5):
    print(i)

A função range() é comum para gerar sequências de números, ideal para laços baseados em contagem.

Exemplo 3: Estrutura while


contador = 0

while contador < 3:
    print("Contagem:", contador)
    contador += 1 # Incrementa o contador

O laço while continua executando enquanto sua condição (contador < 3) for verdadeira. É crucial garantir que a condição eventualmente se torne falsa para evitar um loop infinito.
