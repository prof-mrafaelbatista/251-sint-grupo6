Estruturas de Seleção em Python

Conceito

As estruturas de seleção (ou condicionais) permitem que um programa tome decisões e execute diferentes blocos de código com base em condições. Elas avaliam uma expressão lógica (que resulta em verdadeiro ou falso) e, dependendo do resultado, escolhem qual caminho seguir no fluxo do programa. As principais estruturas são `if`, `if-else` e `if-elif-else`.

Aplicação

Estruturas de seleção são amplamente utilizadas para:

  • Validação de Dados: Verificar se a entrada do usuário é válida (ex: um número está dentro de um intervalo).
  • Controle de Fluxo: Direcionar o programa para diferentes funcionalidades com base em escolhas do usuário ou estados do sistema.
  • Lógica de Negócios: Implementar regras e condições que definem o comportamento do programa (ex: calcular desconto com base no valor da compra).
  • Segurança: Autenticar usuários ou verificar permissões de acesso.

Exemplos de Código

Exemplo 1: Estrutura if simples


idade = 18

if idade >= 18:
    print("Você é maior de idade.")

Neste exemplo, a mensagem "Você é maior de idade." só será exibida se a condição idade >= 18 for verdadeira.

Exemplo 2: Estrutura if-else


numero = 10

if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

Aqui, o programa verifica se o número é par. Se for, executa o bloco if; caso contrário, executa o bloco else.

Exemplo 3: Estrutura if-elif-else


nota = 75

if nota >= 90:
    print("Conceito A")
elif nota >= 80:
    print("Conceito B")
elif nota >= 70:
    print("Conceito C")
else:
    print("Conceito D")

Este exemplo demonstra múltiplos caminhos. O programa verifica as condições em ordem e executa o primeiro bloco cuja condição é verdadeira. Se nenhuma das condições if ou elif for verdadeira, o bloco else é executado.
