Vetores e Matrizes em Python

Conceito

Em Python, o conceito de vetores e matrizes é geralmente implementado usando listas. Uma lista pode ser usada como um vetor (uma coleção linear de itens) e uma lista de listas pode representar uma matriz (uma estrutura bidimensional). Embora Python não tenha tipos nativos como "array" ou "matrix" no sentido de outras linguagens (como C++ ou Java), suas listas são flexíveis o suficiente para simular essas estruturas.

  • Vetor (Lista Simples): Uma sequência ordenada e mutável de itens. Pode conter diferentes tipos de dados.
  • Matriz (Lista de Listas): Uma lista onde cada elemento é, por sua vez, outra lista, formando linhas e colunas.

Aplicação

Vetores e matrizes (listas) são usados para:

  • Armazenamento de Dados: Organizar coleções de dados relacionados (ex: nomes de alunos, temperaturas diárias).
  • Processamento de Dados Tabulares: Representar tabelas, como dados de planilhas ou bancos de dados (para matrizes).
  • Gráficos e Imagens: Representar pixels de imagens ou dados para gráficos.
  • Algoritmos: Implementar algoritmos que operam em coleções de dados, como busca, ordenação ou operações matriciais (com bibliotecas como NumPy).

Exemplos de Código

Exemplo 1: Criação e Acesso a um Vetor (Lista)


# Criando um vetor (lista) de números
numeros = [10, 20, 30, 40, 50]

# Acessando elementos (índices começam em 0)
print("Primeiro elemento:", numeros[0])
print("Terceiro elemento:", numeros[2])

# Alterando um elemento
numeros[1] = 25
print("Lista após alteração:", numeros)

Listas são mutáveis, o que significa que seus elementos podem ser alterados após a criação.

Exemplo 2: Criação e Acesso a uma Matriz (Lista de Listas)


# Criando uma matriz 3x3
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Acessando um elemento (linha, coluna)
print("Elemento na linha 0, coluna 1:", matriz[0][1]) # Saída: 2

# Iterando sobre a matriz
for linha in matriz:
    for elemento in linha:
        print(elemento, end=" ")
    print() # Pula para a próxima linha

Acessar elementos em uma matriz requer dois índices: um para a linha e outro para a coluna.
